Welcome to SigniVerse's Repository! 🌠

SigniVerse is a serious educational game designed to support children learning Jordanian Sign Language (LIU). The game employs computer vision to recognize the player's hand gesture and offer real-time feedback. It covers foundational categories such as the *alphabet, numbers, and everyday household items*. Each category is taught through a minigame along with its respective sign recognition model.</p>
